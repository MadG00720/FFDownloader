# FFDownload 1.0.0

Browser-assisted batch downloader and archive extractor.

## Overview

FFDownload is a desktop application that helps you download files in batch and extract archives efficiently. It combines browser automation with archive extraction capabilities to streamline your download workflow.

## Features

- **Batch Downloading** — Download multiple files at once using electron-dl-manager
- **Archive Extraction** — Extract 7z and RAR archives with built-in support
- **User-Friendly Interface** — Simple and intuitive desktop application

## Download

Choose one of the following options to download FFDownload 1.0.0:

### Installer (Recommended)

### Portable Executable

## Installation

### Using the Installer
1. Download the NSIS Installer from the link above
2. Run the installer and follow the setup wizard
3. Launch FFDownload from your Start Menu

### Using the Portable Executable
1. Download the portable executable
2. Place it anywhere you'd like
3. Run the exe directly
